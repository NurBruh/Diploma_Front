# Инструкция по работе с приложением

## Запуск приложения

Приложение успешно запущено и доступно по адресу: **http://localhost:5174/**

## Основные возможности

### 1. Навигационная панель (Header)
- Хлебные крошки для навигации: Главная → Страница Менеджера ОР → Студенты
- Переключатель "Версия для слабовидящих"
- Выбор языка: Қаз, Рус, Eng
- Профиль менеджера

### 2. Фильтры поиска студентов
Доступны следующие фильтры:
- **ФИО студента** - текстовый поиск по имени
- **ИИН студента** - поиск по идентификационному номеру
- **Курс** - выбор из списка (1, 2, 3, 4)
- **Форма обучения** - Очная/Заочная
- **Институт** - выбор института
- **Кафедра** - выбор кафедры
- **Тип гранта** - Государственный, Академический, Ректорский, Лицей

После выбора фильтров нажмите кнопку **"ОТОБРАЗИТЬ"** для применения.

### 3. Инструменты экспорта/импорта
- **Импорт из Excel** - загрузка данных из файла Excel
- **Экспорт** - выгрузка данных в выбранном формате (Excel, PDF, CSV)

### 4. Таблица студентов
Отображает следующую информацию:
- № - порядковый номер
- ФИО - полное имя студента
- ИИН - идентификационный номер
- Курс - текущий курс обучения
- Форма обучения - очная/заочная
- Институт - наименование института
- Кафедра - наименование кафедры
- Тип гранта - с цветовым кодированием:
  - 🔵 Государственный (синий)
  - 🟡 Академический (желтый)
  - 🟣 Ректорский (фиолетовый)
  - 🟢 Лицей (зеленый)
- Статус стипендии - назначено/не назначено
- Расчетный счёт - банковский счет студента
- Кнопка "Принять решение" - для управления стипендией

## Технические детали

### API
Приложение подключено к API: `https://84ec8b151116fab6.mokky.dev/front`

При загрузке страницы автоматически загружаются все студенты из базы данных.

### Фильтрация
Фильтрация происходит локально на клиенте после загрузки всех данных. Это обеспечивает быструю работу фильтров без дополнительных запросов к серверу.

### Состояния загрузки
- При загрузке данных отображается спиннер с текстом "Загрузка данных..."
- Если нет данных для отображения, показывается сообщение "Нет данных для отображения"
